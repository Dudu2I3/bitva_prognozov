# Локальный запуск

## Шаги

1. Python 3.11+ должен быть установлен (`python3 --version`)
2. Создать виртуальное окружение:
   ```
   python3 -m venv venv
   source venv/bin/activate        # Mac/Linux
   venv\Scripts\activate           # Windows
   ```
3. Установить зависимости:
   ```
   pip install -r requirements.txt
   ```
4. Скопировать `.env.example` → `.env` и вписать туда свой токен бота:
   ```
   cp .env.example .env
   ```
   Открой `.env` и заполни `BOT_TOKEN=` своим значением. Файл `.env` не должен попадать в Git.
5. Создать `.gitignore` (если будешь когда-нибудь класть проект в Git):
   ```
   .env
   venv/
   *.db
   __pycache__/
   ```
6. Загрузить расписание матчей: заполнить `matches_seed_template.csv` (или попросить Claude в проекте написать скрипт-загрузчик CSV → БД)
7. Запустить бота:
   ```
   python -m bot.main
   ```
8. В Telegram написать своему боту `/start` — должна прийти регистрация

## Про токен бота

Токен — это пароль к управлению ботом (можно от его имени слать сообщения, менять настройки). Правила:
- Только в `.env`, никогда в коде или файлах, которые куда-то публикуются
- Если случайно попал в публичный репозиторий, скриншот, или куда-то ещё — перевыпусти через `@BotFather` → `/revoke` (займёт 10 секунд, старый токен сразу станет недействителен)

## Дальше — сервер (когда будет нужно)
Локальный запуск работает, только пока твой компьютер включён и скрипт запущен. Для постоянной работы (бот отвечает 24/7) понадобится VPS или сервис типа Railway/Render — это отдельный шаг, не нужен для текущего MVP.
