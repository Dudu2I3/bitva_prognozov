# Битва прогнозов ЧМ-2026

Telegram-бот для приватного пула прогнозов на матчи ЧМ-2026.

## Быстрый старт

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r ../requirements.txt
cp .env.example .env   # вписать BOT_TOKEN и ADMIN_TELEGRAM_ID
python -m bot.main
```

Подробнее — в `../docx/05_SETUP_LOCAL.md`.
